# VrMPIT
Итоговый проект состоит из 2х частей
Первая часть это прграмма для врача терапевта. В самой программе есть управление пациентами ( назначение приемов, выгрузка данных и т.д ).
Кроме того в программе терапевт может полностью контролировать процесс с помощью панели управления сценой. 
Врач может: включать паузу, переводить пациента в EMDR и Безопасное место, получать все логи с самой гарнитуры. Так же реализован функционал просмотра картинки с самого устройства, что поможет врачу наиболее точно определять раздражители у пациента.

Вторая часть это само автономное приложение на устройство. В нем пациент просмотравает сцены и может управлять переходом в различные модули. Но с помощью панели терапевта лечение становится намного легче т.к врач может полностью управлять приложением.

Шлем подключается к пк в локальной сети автоматически. Не нужно каждый раз указывать IP адресса. Достаточно сначала запустить программу терапевта, а потом приложение на гарнитуре ( quest 3 ) и гарнитура сама подключится к врачебной программе.
